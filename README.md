# HandsOn1  
**태그로 스킬 관리하기**

GAS에서는 태그의 추가/제거으로 능력 사용을 제어할 수 있습니다.  
점프 금지 존(트리거)에 들어가면 `Zone.Stop.Jump` 태그가 캐릭터에 붙는데, 해당 태그는 **점프 Ability의 Activation Blocked Tags**에 설정되어 플레이어는 점프를 할 수 없습니다.
